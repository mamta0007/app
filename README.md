# 🚀 AI-Powered Career Guidance & Interview Preparation System

A FastAPI-based backend system that helps users with **resume analysis, job description parsing, interview preparation, question generation, and career roadmap creation** using AI.

---

## 📌 Features

### 🧠 Resume Analysis
- Extract skills, experience, and technologies
- Identify strengths and weaknesses
- Suggest improvements

---

### 📄 Job Description (JD) Analysis
- Parse job requirements
- Match candidate skills with JD
- Identify skill gaps

---

### 🎯 Interview Preparation
- Generate mock interview questions
- Provide structured interview practice
- AI-based preparation support

---

### ❓ Question Generation Module
- Generates interview questions from:
  - Resume
  - Job Description
- Covers:
  - Technical questions
  - HR/Behavioral questions
- Difficulty-based question generation (Easy / Medium / Hard)

---

### 🛣️ Career Roadmap Generator
- Personalized learning path
- Step-by-step skill roadmap
- Goal-based progression plan

---

### 📊 Report Generation
- Combined AI analysis of resume + JD
- Skill gap report
- Career improvement suggestions
